# 🌐 Language Translation Tool

A simple and attractive Language Translation Tool built with Python and Tkinter.

## 🚀 Features
- Translate text to 100+ languages
- Clean dark-themed UI
- Auto-detects source language
- Supports Hindi, Telugu, French, Spanish, Japanese and more

## 🛠️ Technologies Used
- Python 3.13
- Tkinter (GUI)
- Deep Translator (Google Translate API)

## 📦 Installation
pip install deep-translator

## ▶️ How to Run
python translator.py

## 🌍 Language Codes
- Hindi → hi
- Telugu → te
- French → fr
- Spanish → es
- Japanese → ja
- Arabic → ar
